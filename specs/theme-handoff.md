# 主题页原型 — MiniMax M3 设计规格书

## 设计目标

做一个每次打开网站都会播放的开场主题页。风格参考 `/Users/jojo/Pictures/微信图片_20260607145834_65_2.jpg`——夕阳都市、暖橙色调、有人物伸手的动漫感。

## 技术栈

- Three.js（CDN 引入）— 3D 场景
- GSAP（CDN 引入）— 动画编排
- 单文件 HTML 原型

## 交互流程

```
打开页面
  → 暖橙色 3D 城市夕阳场景，背景跟随鼠标移动产生视差
  → 画面中央一个按钮：「开始旅途」
  → 点击按钮
  → 动画播放：人物从画面中伸出手，拉住观众的手，带着往前走
  → 白光一闪（全屏白色渐入渐出）
  → 进入角色选择页（本原型里先跳到一个占位页面，后续集成时替换）
```

## 技术要求

### 鼠标视差
- 鼠标移动时，3D 场景产生深度视差效果（前景人物/建筑跟鼠标位移大，远景天空跟鼠标位移小）
- 移动平滑，不抖动

### 按钮
- 画面正中央
- 文字「开始旅途」
- 暖色调，融入整体设计
- hover 有呼吸光效

### 动画序列（GSAP timeline）
1. 点击按钮 → 按钮消失
2. 人物手臂伸出，手掌向观众（0.8s）
3. 视角拉近，仿佛被拉着向前走（1.5s）
4. 白色闪光遮盖全屏（0.3s）
5. 过渡到目标页面

### 3D 场景元素建议
- 远景：城市天际线剪影，夕阳天空球
- 中景：建筑群
- 近景：人物模型（至少上半身 + 手臂）
- 粒子：漂浮的光点/尘埃增加氛围

### 设计约束
- 必须用 onclick，不用 addEventListener
- 禁 emoji
- 字体 system-ui / PingFang SC
- 无 AI 套话文案

## 交付物

1. `/Users/jojo/projects/intern-growth-nav/prototype/theme.html` — 独立可运行的原型
2. 加载 Three.js 和 GSAP 的 CDN 链接（不要内嵌库文件）
3. 点击「开始旅途」播放完整动画 → 动画结束后显示一个占位文字「进入角色选择」（后续集成时替换）
4. 必须在浏览器实测通过
